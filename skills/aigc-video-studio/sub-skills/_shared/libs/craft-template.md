# CRAFT 提示词模板（设计稿 §6）

> 全技能统一基准：第零层去 AI 味基座 + 四层结构 + CRAFT 八要素校验。

## 第零层 · 去 AI 味基座（强制注入）

在四层结构**之前**强制注入，作为全片质感下限。词库见 `deai-keywords.yaml`（按风格分组）。`prompt_qc.py` 校验缺失则硬阻断。

```
正向强制词：超写实质感, 真实电影摄影, 胶片颗粒, 自然皮肤纹理/真实材质,
            真实光线衰减与阴影, 镜头呼吸/轻微手持感, 景深与焦外自然
反向强制词：no CG look, no plastic skin, no over-smooth/磨皮, no uncanny valley,
            no over-saturated AI colors, no waxy texture, no perfect symmetry
```

```

【第一层 · 基础设定】

【第二层 · 氛围与画质】

【第三层 · 分镜执行】

```

## CRAFT 八要素校验公式

`[去AI味质感] + [主体] + [动作] + [背景/环境] + [视觉风格] + [相机/摄影] + [构图坐标] + [情绪/氛围]`

八要素齐备；**缺失任一即阻断出卡**（结构硬阻断，确定性无噪声）。LLM-as-Judge 在结构校验通过后再按八要素语义评分（0–100），仅排序/标红，不阻断。

八要素键名（供 `prompt_qc.py` 引用）：

```
deai_texture     # 去AI味质感
subject          # 主体
action           # 动作
environment      # 背景/环境
visual_style     # 视觉风格
camera           # 相机/摄影
composition      # 构图坐标
mood             # 情绪/氛围
```

## 行为逻辑分层模板（设计稿 §6.2，S-P1-5）

> 分镜层（第三层）的「动作」由行为逻辑增强器按**戏种**分层填充，动机优先取自
> `CharacterCard.feature_card.behavior_traits`。`compile_genspec.build_behavior_logic` 实现此模板。

**基础规则集（R1–R5）**

| 规则 | 含义 | 示例 |
|---|---|---|
| R1 动机补全 | 每个主要动作挂「因为/为了」从句 | 狂奔 → 狂奔时压住帽子（怕被风吹掉） |
| R2 情绪因果 | 情绪表达给原因 | 哭 → 看到烧毁的合影后落泪 |
| R3 物理链条 | 多动作按因果链排序 | 听到声→停步→压帽檐→手移向枪 |
| R4 留白声明 | 非关键细节显式留白，不锁死每一帧 | 周围废墟细节由模型自由发挥 |
| R5 素材来源 | 动机优先取自 CharacterCard.behavior_traits | 帽子怕吹掉 ← accessories 注释 |

**戏种分层（统一为：处境层 → [动作层 \| 情绪层] → 因果层 → 留白层）**

| 戏种(drama_type) | 分层结构 | 要点 |
|---|---|---|
| 动作戏 | 处境层（威胁/约束）→ 动作层（招式/位移）→ 因果层（为何此动作）→ 留白层 | 强调物理链条与空间关系 |
| 情感戏 | 处境层（关系/记忆触发）→ 情绪层（外显微表情）→ 因果层（情绪来源）→ 留白层 | 强调情绪的「可见因」 |
| 追逐戏 | 处境层（追/逃方位）→ 动作层（节奏变化）→ 因果层（地形/障碍驱动）→ 留白层 | 强调连续运动与镜头跟随 |

shotlist 镜头可标 `drama_type`，编译时套用对应模板；缺省按「动作戏」。
留白层（R4）落地「随机性即资源」：刻意保留即兴空间。

## 坐标化构图模板（设计稿 §6.2 分镜层，S-P1-5）

> `compile_genspec.build_composition_block` 把 `composition` 规范化注入分镜层。
> shotlist 的 `composition` 字段建议用坐标化写法，缺省回退「主体置三分线交点」。

```
三分法：主体置三分线交点（左上/右上/左下/右下其一），视线/动势指向对侧留白
对角线：主体沿画面对角线排布，强化纵深与动势（如左下→右上）
画面分割：前景/中景/背景按比例分层（如前景占左 1/3 遮挡，主体居中右）
中心构图：主体居中（仅用于仪式感/对峙强调，慎用以免呆板）
```
