# 镜头语言库（shot-language，设计稿 §5 SK4 / §6）

> 景别 / 运镜 / 构图词典。SK4 取 camera_move，SK5 据 `camera-preset-map.yaml` 翻译为平台预设。

## 景别（shot_size）
- 远景（extreme wide）：交代环境
- 全景（wide）：人物与环境关系
- 中景（medium）：叙事主力
- 近景（close）：情绪
- 特写（close-up）：细节/强情绪
- 大特写（extreme close-up）：眼神/道具

## 运镜（camera_move，对应 camera-preset-map）
- 缓慢推轨 / 拉远 / 环绕跟随 / 手持跟拍 / 甩镜 / 急速变焦 / 升降镜

## 构图（坐标化写法模板）
- 三分法：`主体居右三分纵线`
- 对角线：`视线导向左上对角`
- 前景遮挡：`前景油桶剪影`
- 中心构图：`主体居中，对称弱化`

## 时长节奏
- 快段 2–4s/镜；慢段 6–8s/镜；每镜 ≤15s。
