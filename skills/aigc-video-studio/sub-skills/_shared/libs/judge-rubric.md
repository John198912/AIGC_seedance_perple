# LLM-as-Judge 锚点校准 + 两级处理规则（设计稿 §5 SK5 / §6.1，S-P1-3）

## 两级处理

1. **结构级（硬阻断，确定性）**：CRAFT 八要素缺项 → `structural_blockers[]`，直接阻断出卡。
2. **语义级（仅排序/标红，不阻断）**：八要素齐备后按语义评分 0–100：
   - `craft_score`（建议阈值 ≥75）
   - `deai_score`（去 AI 味强度，建议阈值 ≥70）
   - `rubric_anchor_drift`（单锚点偏差，>15 告警漂移）

## 锚点样本（防漂校准，占位）

> 后续阶段填充实际锚点样本（已校准的"高分/中分/低分"提示词样例），供 Judge 对齐评分尺度，避免分数漂移。

| 锚点 | craft_score | 说明 |
|---|---|---|
| 黄金样本 | 92 | 八要素齐备、坐标化构图、行为逻辑完整、去 AI 味强 |
| 及格样本 | 75 | 八要素齐备但构图/动机较弱 |
| 不及格样本 | 50 | 要素勉强齐备、表述笼统（仅排序，不阻断） |

## 防漂规则

- 每次评分附带锚点对照，计算 `rubric_anchor_drift`；>15 时告警并要求人工复核 Judge 输出。
- Judge 只排序不决策审美；G3/G6 人工 Gate 永不被替代。
