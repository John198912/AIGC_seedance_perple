# SK2 · screenplay-writer（编剧 / 世界观）

> 把项目蓝图展开为可拆镜的分场剧本与情绪曲线。详见设计稿 §5 SK2。

## 触发 / 输入 / 输出

- 触发：G1 通过、进入 S2_SCRIPT。
- 输入：`brief.md` + `style-bible.md`。
- 输出：`02_screenplay/screenplay.md`（C3）；series 模式另产 `episodes/ep-NN.md`。

## Gate

**G2 剧本确认**。

## 职责

产出梗概、分场剧本、情绪曲线；剧集模式产出多集弧线与跨集设定圣经。

## 提示词要点

1. **短视频叙事约束硬编码**：前 3 秒钩子、15–30 秒一个情绪节拍、结尾记忆点。
2. 每场标注地点 / 时间 / 在场角色 / 核心动作 / 情绪值(0–10) / 预估镜头数。
3. **情绪曲线表格 + 校验"无连续 3 场平值"**。
4. `character_briefs` 按 C4 `feature_card` 预格式化。
5. **全程无台词/少台词优先**（音画叙事，规避口型风险，呼应《丧尸清道夫》经验）。

## 脚本

`emotion_curve.py`（分场表 → 情绪曲线 + 连续 3 场平值校验，供 Gate 审阅）。
