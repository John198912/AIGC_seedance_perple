# platform-jimeng · 入口路径树（ui-map）

> verified_at: 2026-06-01（活文档，UI/合规迭代快；SK6 实测与此不符时提示更新）。

## 主入口（中文原生）

- 首页 → `视频生成`：中文提示词直注，文/图生视频。
- `沉浸式短片` 模式：长镜头/连续叙事。
- 角色 / 参考上传：挂参考图保持身份一致。

## 与火山方舟 API 的关系（重要）

即梦与 `channel-api` 的火山方舟（volcano）**同源不同入口**：

- 即梦 = **UI 卡**入口（人工在网页操作）。
- 火山方舟 = **API 卡**入口（`api_adapter.py` 异步调用，凭证走 `VOLCANO_AK` 环境变量）。

二者模型同源但 UI/通道差异（参数表达、画幅枚举、水印规则）在 `quirks.md` 标清。

提示词用**中文**（语言策略：jimeng/volcano→zh）。
